# 추가 문법 관련 설명
!<https://choyunjin.kr/img/KASUMI.jpg>
## 글자색
### 코드
```markdown
&[yellow 내용]
```
### 결과물
<span style="color:yellow">내용</span>
## 매크로(함수)
### 코드
```
@[someMacro(arg1,arg2,...,args)]
```
### 사용예
ruby(bottom,top) - 루비문자 출력
```
@[ruby('修学能力試験','しゅうがくのうりょくしけん')]
```
### 결과물
<ruby>
    <rb>修学能力試験</rb>
    <rt>しゅうがくのうりょくしけん</rt>
</ruby>
